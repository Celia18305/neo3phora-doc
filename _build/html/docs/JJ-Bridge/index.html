区块链中间件¶

JJ-Bridge是一个链接区块链和服务的中台，JJ-Bridge就像是一座桥一样，一边集成了访问区块链的SDK（Typechain），一遍通过httpService的形式把服务暴露给客户使用，让客户就像使用传统服务一样使用区块链服务。
