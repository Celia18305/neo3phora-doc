# 基本介绍和项目目的



## 基本介绍

JJToken旨在打造去中心化的电子卡劵销售，消费，核销新模式，让企业，员工，工会，商家在去中心化的区块链平台下，实现：

1: 专款专用

2: 记录可查

3: 互相监督

4: 便捷管理



### 一、专款专用

企业委托给工会的每一笔Token，都会用专门的账户保存，账户和账户之间相互隔离，规避资金风险。



### 二、记录可查

每一笔交易，在区块链上都会以Transaction的形式表现，所有记录都会在链上展示，并且会备份一份数据到中心化的数据库上。



### 三、互相监督

众所周知，区块链是一个去中心化的账本，所有的交易都会统一的，完备的展示在区块链上，并对所有人公开。方便日后检查和监管。



### 四、便捷管理

在中心化系统中，每次交易都需要人工对账，支付，非常繁琐，代币进行链上对账，更为简洁。
